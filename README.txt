Входные данные для сетки (строго прямоугольной формы)

лямбда бетта гамма 

кол-во узлов по x, первый узел x, последний узел x
кол-во узлов по y, первый узел y, последний узел y

Краевые условия:

Первые краевые:
Номер ку - 1, далее кол-во узлов, в которых выполняется ку, после - список этих узло

Вторые и Третьи краевые: 
Номер ку (сначала 2, если они есть), далее кол-во граней, в которых выполнено ку, после - пары узло, образующих эти грани
(Далее точно так же для третьих, если они есть)
(ОБЯЗАТЕЛЬНО пары записываются так, что узлы идут слева направо или снизу вверх (то есть узлы в паре стоят по возрастанию))
